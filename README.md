# Adarsh Public School Website

This is the official website for Adarsh Public School, built with HTML, CSS, and JavaScript.

## Features

- Responsive school portal
- Student zone with resources
- Gallery of school events
- Book list management
- Admission information
- Admin dashboard
- Notice board

## Files

- `index.html` - Main homepage
- `admin.html` - Admin login page
- `dashboard.html` - Admin dashboard
- `style.css` - Styling
- `script.js` - Interactive features

## Live Site

This site is published on GitHub Pages. Visit the live site at:
https://moviepclip-arch.github.io/shcoolsite/

## Installation

Clone the repository:
```bash
git clone https://github.com/moviepclip-arch/shcoolsite.git
```

Then open `index.html` in your web browser.
